# Website Review: hdsok.com & hds.live

**Review Date:** November 20, 2025

## Executive Summary

Both websites showcase Homeland Development Services LLC with a consistent dark theme and neon green branding. The sites complement each other - hdsok.com serves as the marketing/information hub while hds.live appears to be the application/console portal.

---

## Design System Analysis

### Color Palette

**Primary Colors:**
- **Background:** Dark black/near-black (#0a0a0a or similar)
- **Primary Accent:** Neon green (#00ff41 or #00cc33)
- **Card Background:** Slightly lighter dark (#1a1a1a)
- **Navigation Bar:** Dark grey (#1f1f1f)
- **Text Colors:**
  - Primary text: White (#ffffff)
  - Secondary text: Light grey (#d1d5db / gray-300)
  - Tertiary text: Grey (#9ca3af / gray-400)

**Design Philosophy:** High contrast dark theme with vibrant neon green accents creates a modern, tech-forward aesthetic that's easy on the eyes while maintaining professional credibility.

### Typography

**Observed Characteristics:**
- Large, bold headlines (5xl-6xl sizes)
- Clear hierarchy with multiple font sizes
- Good line-height for readability
- Sans-serif font family (likely Inter, system fonts, or similar)
- Font weights: Regular, Semibold, Bold

### Layout Structure

**hdsok.com Layout:**
1. **Fixed Navigation Bar** (top)
   - Logo on the left
   - Navigation menu items centered/right
   - Dark grey background
   - Sections: Home, Contact Us, About HDS, Services (with dropdown), Vendors, HDS Console
   - Customer Login link (external to cmms.hdsok.com)

2. **Hero Section**
   - Full-width dark background
   - Centered content with large headline
   - Subheading text
   - CTA buttons (primary green, secondary outlined)

3. **Content Sections**
   - Max-width containers (approximately 1280px)
   - Ample padding and spacing
   - Alternating content layouts

4. **Footer**
   - Multiple columns
   - Navigation links
   - Contact information
   - Service links
   - Copyright and legal links

## Reusable Design Elements

### 1. Color Variables
```css
--dark-bg: #0a0a0a
--dark-card: #1a1a1a
--grey-nav: #1f1f1f
--neon-green: #00ff41
--neon-green-dark: #00cc33
--text-white: #ffffff
--text-gray-300: #d1d5db
--text-gray-400: #9ca3af
```

### 2. Button Classes
```css
.btn-primary {
  background: neon-green;
  color: dark-bg;
  padding: 0.75rem 2rem;
  border-radius: 0.5rem;
  font-weight: 600;
  transition: all 0.3s;
}

.btn-primary:hover {
  background: neon-green-dark;
}

.btn-secondary {
  border: 2px solid neon-green;
  color: neon-green;
  background: transparent;
  padding: 0.75rem 2rem;
  border-radius: 0.5rem;
  font-weight: 600;
  transition: all 0.3s;
}

.btn-secondary:hover {
  background: neon-green;
  color: dark-bg;
}
```

### 3. Card Styles
```css
.card {
  background: dark-card;
  border-radius: 0.5rem;
  padding: 2rem;
}

.card-bordered {
  border: 4px solid neon-green;
  background: dark-card;
  border-radius: 0.5rem;
  padding: 2rem;
}
```

### 4. Section Spacing
- **Vertical padding:** py-16 to py-20 (4rem to 5rem)
- **Horizontal padding:** px-4 (1rem) on mobile
- **Max width containers:** 1280px (max-w-7xl) or 1152px (max-w-6xl)
- **Gap between elements:** 4-6 spacing units

### 5. Typography Scale
- **Hero headline:** text-5xl md:text-6xl (3rem-4rem)
- **Section headlines:** text-4xl (2.25rem)
- **Subheadings:** text-xl to text-2xl (1.25rem-1.5rem)
- **Body text:** text-lg (1.125rem)
- **Small text:** text-base (1rem)
